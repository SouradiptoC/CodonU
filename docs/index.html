Welcome to CodonU’s documentation!

Introduction

CodonU is an integrated package for Codon Usage Analysis. This is written by Souradipto Choudhuri in 2023 and was his final year project in Biotechnology major. This is only the documentation of the project.

Please find the examples in the git repo of the project if you want one.
